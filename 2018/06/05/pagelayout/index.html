Page layouts by CSS

Posted on Jun 5 2018 | In FrontEnd

Page layouts

When discussing a page layout, I was embarrassed because I did not really learned that formally. I learned to create fixed, responsive web page layout by using float, min-width, percentage…etc, but I never learned the concepts and have no idea what is adaptive, fluid layouts, I think I need a big picture so I went looking.

Nick said: “In short, each page layout name describes how the layout behaves when the page is viewed at different browser widths. Because the web page is been viewed from different devices width different width.”

There are total four types of page layout: static, fluid, adaptive, responsive.

Static Page Layout

Static page layout, also called fixed width layout. So the elements has a fixed preset width no matter how you resize the browser. This is not commonly used but it’s a good start to learn web layouts.

Liquid Page Layout

A liquid page layout, also called fluid width layout. Basically a liquid layout will use percentages instead of pixels, so the container will always occupy that much percentage space on the screen. But there are some major drawbacks at very large or very small browser widths. For example, if the browser is very large, some container could be stretched too far, a single paragraph might run across the page on a single line. Conversely, a paragraph on a small screen could be too crowded.

Adaptive Page Layout

A adaptive page layout means there are several versions of layout displayed based on screen size of the viewer. It is achieved by use CSS media query to detect the width of the browser and apply CSS rules accordingly. The benefit is that the page will load faster since it uses mobile first approach, which means browser will only load the additional CSS when your browser reach the break point width.

Responsive Page Layout

Responsive page layout uses both liquid and adaptive layout design. There are several break-points to separate your layouts for different browser/device width. As a browser increase or decrease, the elements will stretch or shrink accordingly.
