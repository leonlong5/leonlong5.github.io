Centering in CSS

I’ve been confused by this problem for long time when I was new to CSS, so let’s take a look here to make it clear and easier.

Horizontally

  • text-align: center

    This is the rule to horizontally align inline elements(like p, span) int the center. Usually used to centering text.

  • margin: 0 auto
    This is widely used to centering a block element, usually the entire container of a section.

Vertically

  • line-height
    This usually used to set a single line vertically center

  • vertical-align

    If the text has multiple lines, we can set the parent container as a table cell, then use vertical-align rule to set the element vertically center.

Both horizontally and Vertically

  • Use transform
    We know the top, left will set the element left top corner of an element to the position, by using transform to move the element back 50% of its original position can help us align the element both horizontally and vertically center.

  • Use flexbox

  • Use grid
