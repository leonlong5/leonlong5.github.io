Commonly used commands

Commands in linux system are widely used in our daily work nowadays, so let’s take look at the most commonly used commands, refresh memory and pick up some learned but never used commands to help boost our productivity.

Terminal navigation commands

ls - Display a list of files and folders under current directory

cd <directory> - move in to the directory

pwd - Displays the current terminal directory as an absolute path

history - Displays a list of recently used commands

File management commands

cat <fileName> - display contents of the file

touch <fileName> - create a new files

cp <from> <to> - copy the file from current directory to target directory

mv <from> <to> - move the file from current directory to target directory

find - Searches a specific directory to find files that match a given set of criteria

grep - Searches a specific file or set of files to see if a given string of text exists, and if it does, tell you where the text exists in those files.

mkdir <directoryName> - create a new directory

rmdir <directoryName> - remove the directory

rm <fileName> - remove file, be careful this will remove the file from your hard drive forever.

System management commands

df - displays how much space is used and free on your System

top - displays a list of currently running processes, sort by how much CPU each processes uses.
