React Native Flat List demo

Posted on Jul 1 2018 | In Mobile

React native Flat List

Recently I worked on a small APP with React Native in dora hackathon, find it is really fun to write Apps with react, so I dive a little bit in it these days, today let’s take a look at a Flat List demo to see how to render a list in React Native.
Demo address: https://snack.expo.io/@leonlong/flatlistdemo

Initialization

First we need to initialize a react native project, I used create-react-native-app to initialize this demo, as you can see from the code we imported the components we needed from react-native

Then we initialize the cities data for the demo and created a class from react component, this is very similar to react, if you are familiar with react before, this will be easy for you to understand. Below is the basic example of a react native component.

Render the list

Now we can start create a flat list by using the Component FlatList.

It has some props for us to use, I will introduce the basics, for more you can refer to https://facebook.github.io/react-native/docs/flatlist

data

We prepared some data for us to use, an array of cities, each object contains city name and population.

Now we can add the data to flat list.

Usually we will put this in state for easy manipulation for later.

Next we use the renderItem to render the list of items.

also created the function to do the job.

Pull down reloading

Now we have the list of cities in the App

Next we will add realoding when user swipe down at the top of the screen for the list.
we will use RefreshControl component to build this feature.

title

The text showing when reloading

tintColor

The color of the animated circle

refreshing

Set this to true when waiting for new data from a refresh

onRefresh

the function that invoked when refresh been triggered by user

In the loadData funciton, it will first set the isLoading to true, to let the animated circle woking to let user know it is loading, then you will need to fetch the data from API to update the data, here for demostration we just reversed the array, last don’t forget to set the isLoading to false, otherwise the waiting circle will keep spining.

Reach end loading more

Our last step is to use ListFooterComponent to build a lazy loading for the list. First we need to add ListFooterComponent and onEndReached to FlatList.

ListFooterComponent

It will be used to build the loading footer component while user is waiting.

onRefresh

This function will be invoked when user reached the end of the page.

In the genIndicator function, we returned a View with ActivityIndicator component, it has some customized attributes for us to use.

In the loadData function, we need to add an argument to distinguish refreshing and loading more, when loading more is triggered, for demonstration purpose, we will append the CITIES to the end of the list.
